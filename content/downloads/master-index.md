# AutoPrompt Kit 2026 - Master Index

## How to use this kit

1. Pick a prompt from a pack that matches your task.
2. Replace placeholders wrapped in <...>.
3. Run it in your preferred model (Claude, Cursor, Grok, etc.).
4. Iterate with the QA checklist.
